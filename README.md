# tpyescript-next-redux-antd
## Feature
- Next.js
- Typescript
- Antd
- Redux / Saga / Persist
- Styled-Components
- Jest

## Installation
```
  git clone https://github.com/juicyjusung/tpyescript-next-redux-antd my-project
  cd my-project
  npm i  
```

## Run
### Devlopment
```
  npm run dev
```
### Serve
```
  npm run build
  npm run start
```
